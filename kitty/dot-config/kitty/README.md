# Experimental
This config is experimental. I have not and likely will not use it heavily.

It came about while playing around with [chafa](https://github.com/hpjansson/chafa/).
I looked at Sixels and Kitty graphics formats for terminal emulators. Since
my go-to terminals do not have Kitty graphics I took a look at Kitty itself.

This setup mostly matches my foot setup, though the fonts are not as I
expected (some spacing and icons issues/differences).
I have not configured Kitty's "advanced" features like tabs, sessions,
remote control, etc.
